How to run:
1) Change config setting in the config directory -> default.json to your settings.
2) Select using mysql by setting "useMySqlAsDatabae"="true"
    or use Dynamodb by setting it "false".

3) Open command prompt in this directory
4) npm install
5) node app.js
6) Surf to localhost:3000
